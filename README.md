# Rubber Duckies FTW Secure E-Commerce Website

## Overview

This project was developed for my Secure E-Commerce course at the University of North Texas. The goal of the project was to create a functional e-commerce website and apply security features that are important for protecting online stores, customer accounts, and payment workflows.

The website, called **Rubber Duckies FTW**, was built as a mock e-commerce store that sold custom rubber ducks and related products. The project focused on both the business functionality of an online store and the security controls needed to support safer e-commerce operations.

## Project Purpose

The purpose of this project was to:

- Design and build a functional e-commerce website
- Add security features commonly used in online stores
- Configure account protection controls
- Support secure checkout behavior
- Apply SSL/TLS protection
- Document the site design and data flow
- Present the project as a secure e-commerce implementation

## Security Features

The project included several security-focused features.

### Two-Factor Authentication

Two-factor authentication was added to strengthen account security. This helps protect user accounts by requiring an additional verification step beyond only a password.

Security value:

- Reduces risk from stolen passwords
- Adds an extra layer of login protection
- Supports safer account access

### Secure Payment Gateway

A secure payment gateway was included to support safer checkout behavior. Payment processing is one of the most important security areas in an e-commerce site because it involves sensitive customer and transaction information.

Security value:

- Keeps payment handling separate from the main website
- Reduces risk of directly exposing payment data
- Supports safer transaction processing

### SSL Certificate

An SSL certificate was used to support encrypted communication between users and the website.

Security value:

- Enables HTTPS
- Helps protect data in transit
- Reduces risk of information being intercepted
- Improves user trust in the website
